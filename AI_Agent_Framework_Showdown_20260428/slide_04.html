🏭 VoltAgent

架构亮点

  • 框架无关 Server 层（Hono/Elysia 可替换）
  • MCP 双角色（Client + Server）
  • A2A 协议（Agent 互操作）
  • 声明式 Workflow 链式 API
  • 6 种 Memory 后端适配器
  • 15+ 种 RAG Chunker

代码示例

new VoltAgent({
  agents: {assistant},
  server: honoServer(),
  workflows: { wf },
});

createWorkflowChain({id,input,result})
  .andThen(execute)
  .andAgent(prompt, agent)
  .andWhen(condition)
