## 核心结论
- **LangChain**：Python 生态之王，研究/原型首选，但 TypeScript 版本弱势
- **VoltAgent**：Agent 平台化路线，MCP+A2A 协议独特
- **Mastra**：全栈化路线，Observational Memory + Studio 时间旅行是杀手级功能，22k stars
